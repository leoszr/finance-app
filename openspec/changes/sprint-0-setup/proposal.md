# Sprint 0 Setup

## Why
Estabelecer a base tecnica do produto para permitir evolucao segura nas sprints seguintes.

## What Changes
- Inicializacao padrao do app com Next.js 14 e TypeScript strict.
- Configuracao do Supabase (schema inicial, RLS e auth Google).
- Fluxo de autenticacao completo (login, callback, middleware e logout).
- Layout base mobile-first com navegacao inferior.
- Configuracao de PWA e pipeline inicial de deploy.

## Capabilities
- authentication

## Impact
- Bloqueia risco de retrabalho estrutural nas sprints funcionais.
- Viabiliza desenvolvimento incremental orientado por OpenSpec.
